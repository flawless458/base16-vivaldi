# B16 for Vivaldi

## Follow the instructions below to apply the theme.

### Import the theme

1.  Download & unzip [master_themes.zip](https://github.com/flawless458/base16-vivaldi/releases/tag/v0).
2.  Open vivaldi **Settings > Themes** & click **Import** a theme.
3.  Enjoy!
